# Abraham-mod 🚀
Kani waa Project-ka Mod Menu ee Android, kaas oo lagu dhisay ImGui iyo C++.

## Sifooyinka (Features):
* **ESP Visuals:** Line, Box, iyo Health.
* **Aimbot:** Si toos ah loogu beegsado cadawga.
* **UI Qurxoon:** Menu si fudud loo isticmaali karo.

## Sida loo isticmaalo:
1. Soo deji "Source Code-ka".
2. Ku dhis (Build) Android Studio ama AIDE.
3. Ku dhex dur (Inject) ciyaarta aad rabto.
